#  Jokes Viewer App

A simple React + Vite application that fetches random jokes from the FreeAPI and displays them in a clean modern UI.

## 🚀 Features
- Fetch random jokes from API
- Simple modern UI
- Built with React + Vite
- Vanilla CSS
- Component-based structure

## 🛠️ Tech Stack
- React
- Vite
- Vanilla CSS
- FreeAPI

## 📡 API Used
```bash
https://api.freeapi.app/api/v1/public/randomjokes/joke/random
```

## 📂 Folder Structure
```bash
src/
│
├── components/
│   ├── JokeCard.jsx
│   └── JokeCard.css
│
├── App.jsx
├── App.css
├── index.css
└── main.jsx
```

## ⚙️ Installation

Clone the project:

```bash
git clone <your-repo-link>
```



Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

## 🎯 Functionality
- Shows a random joke from FreeAPI
- Click the "New Joke" button to load another joke instantly
